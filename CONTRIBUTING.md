# Contributing

Thanks for helping improve this list.

## Entry format
- `[Name](URL) - Short description.`
- One line per entry.
- Keep descriptions concise and neutral.
- Descriptions must start with a capital letter and end with a period.

## Quality checklist
- Check for duplicates.
- Ensure the link is reachable.
- Keep the list alphabetical within a subsection if possible.

## Submitting changes
- Open a pull request.
- If you add a new top-level section, update `Contents`.
